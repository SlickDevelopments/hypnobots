## Development

Install dependencies:

```bash
yarn install
```

Run any bot using:

```bash
yarn serve bots/naming-cop
```

You can change the port using the `PORT` environment variable:

```bash
PORT=3000 yarn serve bots/naming-cop
```

And test your code:

```bash
yarn test
```
